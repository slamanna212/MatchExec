# Keyboard Shortcuts

Shortcuts are disabled when focus is inside a text input, textarea, or select field.

## Navigation

| Key | Action |
|-----|--------|
| `T` | Go to tournaments |
| `M` | Go to matches |
| `H` | Go to home |
| `?` | Show keyboard shortcuts help |

## Match Detail Page

| Key | Action |
|-----|--------|
| `Ctrl+Enter` / `Cmd+Enter` | Move match to next phase |

Advances the match through its phases in order: Created → Gather → Assign → Battle → Complete.

## Scoring Page

| Key | Action |
|-----|--------|
| `↑` | Select previous map |
| `↓` | Select next map |
| `1` | Blue team wins current map |
| `2` | Red team wins current map |

`1` and `2` are no-ops if the current map is already completed or uses FFA/Position scoring.
